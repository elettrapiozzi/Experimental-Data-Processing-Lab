# Experimental Data Processing Laboratory

**Bachelor in Physics - Università degli Studi di Milano**
**Author:** Elettra Piozzi

# About This Repository
This repository contains a collection of **C++** scripts and programs developed during the "Laboratory of Numerical Processing of Experimental Data" course. The exercises focus on applying Object-Oriented Programming (OOP) principles to solve physics problems, analyze datasets, and perform numerical simulations, utilizing standard C++ libraries (STL) and the **CERN ROOT** data analysis framework.

# Covered Topics

**1. Data Analysis & Statistics**
* Computation of statistical estimators (mean, variance, median).
* Dynamic memory allocation and usage of STL containers and ROOT.

**2. Object-Oriented Programming (OOP) in C++**
* Implementation of Encapsulation, Inheritance, and Polymorphism.
* Mathematical operator overloading for physics applications.

**3. Numerical Methods & Integration**
* Root-finding algorithms for mathematical functions.
* Numerical quadrature.
* Pseudo-random number generators and Monte Carlo (MC) integration.

**4. Physics Simulations & Differential Equations**
* Simulation of dipole and gravitational fields.
* Solving differential equations for:
  * Harmonic oscillators.
  * Motion in gravitational fields.
  * Charged particle dynamics in electromagnetic fields.
* Simulation of experimental setups.
